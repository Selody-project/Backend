# Backend

xERN Project

# Getting Started

Clone the repo

```
git clone https://github.com/xERN-shareANDcommunity/Backend.git
```

Create `.env` file:
```
cp .env.sample .env
```

Install dependencies:

```
npm install
```

Run the application:

```
npm run dev
```

## Running Using Docker
```
docker compose up
```
